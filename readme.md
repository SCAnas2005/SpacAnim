# SpacAnime

SpacAnime est un jeu de tir spatial (Space Shooter) en 2D développé en Python. Pilotez votre vaisseau, évitez les champs d'astéroïdes, luttez contre la gravité des trous noirs et survivez le plus longtemps possible dans l'espace infini.

## Prérequis

Pour lancer ce jeu, vous devez avoir **Python 3** installé sur votre machine.

### Important : Installation de Pygame

Ce jeu dépend de la bibliothèque **Pygame**. Vous devez impérativement l'installer pour que le jeu fonctionne.

Ouvrez votre terminal (ou invite de commande) et exécutez la commande suivante :

```bash
pip install pygame
